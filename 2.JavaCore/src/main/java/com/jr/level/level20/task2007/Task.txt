Как сериализовать JR?

Сделай так, чтобы сериализация класса JR была возможной.


Требования:
1.	Класс JR должен существовать внутри класса Solution.
2.	Класс JR должен быть статическим.
3.	Класс JR должен быть публичным.
4.	Должна быть возможна сериализация класса JR.
